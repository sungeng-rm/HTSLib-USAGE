# HTSLib Usage
## Introduction
This project is going to show the basic usage for HTSLib, Containing Bam I/O, VCF/gVCF I/O. The main work will be updated continuously in the next period of time.
## Usage
cd ./htslib<br> 
autoreconf -i<br> 
./configure<br>
make<br>

mv libhts.so.3 ../src<br>  
cd ../src<br>
make<br>
